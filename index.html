What is CSS?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describe how HTML element are to be displayed on screen,paper,or in other media.

  3. Css is not neccesary CSS saves a lot of work, It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline-by using the style attribute inside HTML elements1
  • Internal-by using a style element in head section
  • External-by using a link element to limk to an external CSS file
  • The most common way to add CSS,is to keep the styles in externl CSS files.However,in this tutorial we will use inline and internal styles
    • because this is easier to demonstrate,and easier for you to try it yourself.

When comparing CSS class vs ID,the difference is that CSS applies a style to multiple elements.ID,on the other hand,applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS ,selectors are used to target a specific element or range of the element on a web page. Once an element has been targeted,a style or set of style can be applied to the element.

There is a wide range of selectors available .Two of the commonly used are class and ID Both are used to target elements to which a style should be applied.
